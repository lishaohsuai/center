我是垂直居中
	
	用pisition:absolute  实现
	.parent{width:6em;height:500px;}
	.child{width:100%;}

	.parent{
		position: relative;
	}
	.child{
		position: absolute;
		top: 50%;
		transform: translateY(-50%);
	}

tablecell居中
	table-cell 实现
	.parent1{width:200px;height:300px;}
	
	.parent1{
		text-align: center;//文字居中
		display: table-cell;
		vertical-align: middle;//垂直居中
	}
	.child1{
		display: inline-block;
	}

绝对定位全居中
	绝对定位  不必多说
	.parent2{
		position: relative;
	}
	.child2{
		position: absolute;
		left: 50%;
		top: 50%;
		transform: translate(-50%,-50%);
	}
